## Quick Start

Get started with Isar Plus in minutes:

```dart
// 1. Add to pubspec.yaml
dependencies:
  isar_plus: ^4.0.0
  isar_plus_flutter_libs: ^4.0.0

dev_dependencies:
  build_runner: ^2.4.0

// 2. Define your schema
@collection
class User {
  Id id = Isar.autoIncrement;
  String? name;
  int? age;
}

// 3. Open Isar and use it
final isar = await Isar.open([UserSchema]);

final user = User()
  ..name = 'John'
  ..age = 25;

await isar.writeTxn(() async {
  await isar.users.put(user);
});
```
